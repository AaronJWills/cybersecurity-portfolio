## Useful Commands / Tools
- ping, tracert, nslookup and netstat
- Netstat switches -a -b -o -n
- Dir switches /a /s
- Tree visually shows folders 
- Type (can be used with more)
- Erase (same as del?)
- Tasklist (like task manager) eg. Tasklist /FI "imagename eq sshd.exe"
- Taskkill - eg. Taskkill /PID 4567
